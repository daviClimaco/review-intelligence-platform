# Review Intelligence Platform

## Goal

Create a platform capable of collecting, analyzing and visualizing customer reviews.

## MVP

- Collect Google Maps reviews
- Store reviews
- Categorize reviews
- Sentiment analysis
- Dashboard with metrics and insights

## Tech Stack

- Python
- PostgreSQL
- FastAPI
- Streamlit
- Docker

## Future Features

- Internal reviews
- Doctoralia integration
- AI assistant
- RAG
- Advanced analytics
